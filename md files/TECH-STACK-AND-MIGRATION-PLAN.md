# WordPress to React Migration Plan
## ACBF RSA Website - Technology Stack & Phased Implementation

**Last Updated:** 2025-11-11
**Status:** Approved - Ready for Implementation

---

## Executive Summary

This document outlines the approved technology stack and phased migration strategy for converting the ACBF RSA WordPress website to a modern React application.

### Phased Approach Strategy

**Why Phased?**
- âœ… Get the website live faster (frontend first)
- âœ… Reduce initial complexity
- âœ… Add CMS capabilities later when needed
- âœ… Lower risk - each phase is independent

**Phase 1 Focus:** Beautiful, fast React frontend (4-6 weeks)
**Phase 2 Focus:** Admin panel for content management (2-3 weeks)

---

## Technology Stack (APPROVED)

### Frontend
- **Framework:** React 18+ with Hooks
- **Build Tool:** Vite (fast development & builds)
- **Routing:** React Router DOM v6
- **Styling:** Tailwind CSS (utility-first)
- **State Management:** React Context API + Hooks
- **Data Fetching:**
  - Phase 1: Static JSON imports
  - Phase 2: Axios or Fetch API for Strapi
- **Forms:** React Hook Form
- **Animations:** Framer Motion
- **Carousel/Slider:** Swiper.js
- **Icons:** React Icons (Heroicons style)
- **Date Handling:** date-fns

### Backend (Phase 2 Only)
- **CMS:** Strapi v4 (Node.js-based Headless CMS)
- **Runtime:** Node.js 18+ LTS
- **Database:** SQLite (Phase 2 start) â†’ PostgreSQL (production)
- **API:** REST API (auto-generated by Strapi)
- **Authentication:** Strapi built-in (JWT)

### Hosting (Your Infrastructure)
- **Frontend Hosting:** Your cPanel (existing domain)
- **Backend Hosting (Phase 2):** Your cPanel Node.js support
- **Database (Phase 2):** cPanel MySQL or PostgreSQL
- **Domain:** Your existing domain (already owned)

### Development Tools
- **Version Control:** Git + GitHub
- **Package Manager:** npm or yarn
- **Code Quality:** ESLint + Prettier
- **Testing:** Jest + React Testing Library (optional initially)

```javascript
// src/pages/Portfolio.jsx
import { useState } from 'react';
import { motion, AnimatePresence } from 'framer-motion';
import { usePortfolio } from '../hooks/usePortfolio';

export default function Portfolio() {
  const allItems = usePortfolio();
  const [activeCategory, setActiveCategory] = useState('all');

  const categories = ['all', ...new Set(
    allItems.flatMap(item => item.categories)
  )];

  const filteredItems = activeCategory === 'all'
    ? allItems
    : allItems.filter(item =>
        item.categories.includes(activeCategory)
      );

  return (
    <div>
      {/* Category Filter Buttons */}
      <div className="flex gap-4 mb-8">
        {categories.map(cat => (
          <button
            key={cat}
            onClick={() => setActiveCategory(cat)}
            className={activeCategory === cat ? 'active' : ''}
          >
            {cat}
          </button>
        ))}
      </div>

      {/* Portfolio Grid */}
      <motion.div
        layout
        className="grid md:grid-cols-3 gap-6"
      >
        <AnimatePresence>
          {filteredItems.map(item => (
            <motion.div
              key={item.id}
              layout
              initial={{ opacity: 0 }}
              animate={{ opacity: 1 }}
              exit={{ opacity: 0 }}
            >
              <PortfolioCard item={item} />
            </motion.div>
          ))}
        </AnimatePresence>
      </motion.div>
    </div>
  );
}
```

```javascript
// src/components/forms/ContactForm.jsx
import { useForm } from 'react-hook-form';

export default function ContactForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  const onSubmit = async (data) => {
    // Phase 1: Use Formspree or similar service
    const response = await fetch('https://formspree.io/f/YOUR_ID', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data),
    });

    if (response.ok) {
      alert('Message sent successfully!');
    }
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)} className="space-y-4">
      <div>
        <input
          {...register('name', { required: 'Name is required' })}
          placeholder="Your Name"
          className="w-full px-4 py-2 border rounded"
        />
        {errors.name && <span>{errors.name.message}</span>}
      </div>

      <div>
        <input
          {...register('email', {
            required: 'Email is required',
            pattern: {
              value: /^[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,}$/i,
              message: 'Invalid email address'
            }
          })}
          placeholder="Your Email"
          className="w-full px-4 py-2 border rounded"
        />
        {errors.email && <span>{errors.email.message}</span>}
      </div>

      <div>
        <textarea
          {...register('message', { required: 'Message is required' })}
          placeholder="Your Message"
          rows="6"
          className="w-full px-4 py-2 border rounded"
        />
        {errors.message && <span>{errors.message.message}</span>}
      </div>

      <button type="submit">Send Message</button>
    </form>
  );
}
```

```javascript
// migrate-to-strapi.js
const fs = require('fs');
const axios = require('axios');
const FormData = require('form-data');

const STRAPI_URL = 'https://api.yourdomain.com';
const API_TOKEN = 'your-api-token-here';

const api = axios.create({
  baseURL: STRAPI_URL,
  headers: {
    'Authorization': `Bearer ${API_TOKEN}`,
  },
});

async function uploadImage(imagePath) {
  const form = new FormData();
  form.append('files', fs.createReadStream(imagePath));

  const response = await api.post('/api/upload', form, {
    headers: form.getHeaders(),
  });

  return response.data[0].id;
}

async function migratePost(post) {
  let featuredImageId = null;

  if (post.featuredImage) {
    featuredImageId = await uploadImage(`./public${post.featuredImage}`);
  }

  const strapiPost = {
    data: {
      title: post.title,
      slug: post.slug,
      content: post.content,
      excerpt: post.excerpt,
      featuredImage: featuredImageId,
      publishedAt: post.date,
      // Map categories and tags separately
    }
  };

  await api.post('/api/posts', strapiPost);
  console.log(`Migrated: ${post.title}`);
}

async function migratePosts() {
  const posts = JSON.parse(fs.readFileSync('./src/data/posts.json', 'utf8'));

  for (const post of posts) {
    await migratePost(post);
  }
}

migratePosts().then(() => {
```

```javascript
function transformStrapiResponse(data) {
  if (Array.isArray(data)) {
    return data.map(item => ({
      id: item.id,
      ...item.attributes,
    }));
  }
  return {
    id: data.id,
    ...data.attributes,
  };
}

    return transformStrapiResponse(response.data.data);

    const response = await api.get(`/posts`, {
      params: {
        filters: { slug: { $eq: slug } },
        populate: '*',
      },
    });
    return transformStrapiResponse(response.data.data)[0];
  },

  getByCategory: async (categorySlug) => {
    const response = await api.get('/posts', {
      params: {
        filters: {
          categories: {
            slug: { $eq: categorySlug },
          },
        },
        populate: '*',
      },
    });
    return transformStrapiResponse(response.data.data);
  },
```

```javascript
import { useState, useEffect } from 'react';
import { postsApi } from '../services/api';

export function usePosts() {
  const [posts, setPosts] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    async function fetchPosts() {

      } catch (err) {
        setError(err.message);
      } finally {
        setLoading(false);
      }
```

## Troubleshooting

### Common Issues & Solutions

**Issue: React Router not working (404 on refresh)**
- Solution: Check `.htaccess` is configured correctly
- Verify mod_rewrite is enabled in cPanel

**Issue: Images not loading**
- Solution: Check image paths are correct (use relative paths)
- Verify images are in `public` folder, not `src`
- Check file permissions (644)

**Issue: Site not updating after upload**
- Solution: Clear browser cache
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Check cPanel File Manager - verify files uploaded

**Issue: Slow performance**
- Solution: Run `npm run build` (not dev server)
- Optimize images (compress, convert to WebP)
- Enable caching in `.htaccess`
- Use CDN for assets (Cloudflare)

**Issue: Node.js app crashes (Phase 2)**
- Solution: Check error logs in cPanel
- Verify Node.js version compatibility
- Check database connection
- Use PM2 for process management: `pm2 logs`

**Issue: CORS errors (Phase 2)**
- Solution: Configure Strapi CORS in `config/middlewares.js`
- Add your domain to allowed origins
- Check browser console for specific CORS error
